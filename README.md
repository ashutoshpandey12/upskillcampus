# Predict Remaining Useful Life (RUL) of Turbofan Engines using Machine Learning

## Project Overview

This project predicts the Remaining Useful Life (RUL) of turbofan engines using machine learning techniques. The objective is to estimate the number of operational cycles remaining before an engine is likely to fail, enabling predictive maintenance and reducing unexpected downtime.

## Features

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Machine Learning model training
- Remaining Useful Life prediction
- Performance evaluation
